The Batch Address Test Tool allows you to check and make sure that VIP is returning the correct polling location for a large sample of addresses. Once feed data has been submitted and ingested, you can upload a CSV with a random sample of 100-300 voter addresses and the polling locations they should return, and the tool will check that the API is returning the correct result for each address. Because of the time required for ingestion, data may not be available to review in this tool immediately after submission to the VIP dashboard.

To ensure that polling location data is accurate, please upload a CSV with two columns:

voter_address, expected_polling_location

For best results, please include a sample of 300-500 addresses from a variety of participating Jurisdictions.

To get started, download the template test address CSV!

Limitations: This tool is only meant for to test accuracy where a single precinct is linked to a single polling location. If precincts are linked to multiple polling locations (ie. vote centers or early vote sites), this tool will not effectively evaluate accuracy.
